# Lightweight CNN-Based Food Freshness Detection System Using Smartphone Images for Sri Lankan Market Environments

## Project Description

This research aims to develop a lightweight deep learning system capable of detecting the freshness of fruits and vegetables using smartphone images captured under real Sri Lankan market conditions.

The study compares:
Traditional CNN
MobileNetV2
EfficientNetB0
to identify the most suitable model for future mobile deployment.

## Research Objectives

Collect a custom smartphone image dataset.
Preprocess and augment images.
Train lightweight CNN models.
Compare model performance.
Develop a mobile-friendly food freshness detection solution.

## Technologies

Python
TensorFlow
Keras
OpenCV

## Planned Dataset

A custom dataset will be collected using Android smartphone cameras from Sri Lankan markets, supermarkets, and home environments.
